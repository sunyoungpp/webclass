코딩 - 코드를 입력해서 소프트웨어 제어하는 행위
1) html - 홈페이지에 들어가는 컨텐츠를 입력하는 언어
2) css - 홈페이지의 스타일을 담당하는 언어
3) javascript - 홈페이지의 기능을 담당하는 언어
4) jquery - javascript 언어를 보다 쉽게 사용하기 위해 개선된 언어
흐름
브라우저 - 크롬,익스플로러,사파리,파이어폭스....
클라이언트 - 요청자 요청을 하는 주체->브라우저가 클라이언트에 속한다
서버 - 제공자(저장소)파일들이 저장되는 공간, 클라이언트의 요청에 응답하여 파일을 제공해주는 역할
